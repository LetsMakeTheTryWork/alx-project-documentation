# Use Case Diagram

This diagram visualizes interactions between users and the Airbnb Clone system, showing the main functionalities:

- **User Registration/Login**: Sign up or log in.
- **Property Listing**: Hosts can manage their listings.
- **Property Booking**: Guests can search and book properties.
- **Payments**: Guests can pay securely for bookings.
- **Reviews**: Guests can leave reviews for properties.
- **Messaging**: Guests and hosts can communicate regarding bookings or inquiries.

Actors (Guest, Host, System) and their interactions with these features are illustrated.
